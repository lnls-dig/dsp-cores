  • RST [read/write]: Reset
    Reset input
  • MODE1 [read/write]: Mode Input 1
    Operation mode input of first pair
  • MODE2 [read/write]: Mode Input 2
    Operation mode input of second pair
  • SWAP_DIV_F [read/write]: Swap Divisor
    Divider of clock input
  • CLK_SWAP_EN [read/write]: Clock Swap Enable
    Clock Swap Enable
    write 0: disable switching clock output
    write 1: enable switching clock output

3.7. Windowing Control

HW prefix: bpm_swap_wdw_ctl
HW address: 0x6
C prefix: WDW_CTL
C offset: 0x18

Register to select windowing

31 30 29 28 27 26 25 24
DLY[15:8]
23 22 21 20 19 18 17 16
DLY[7:0]
15 14 13 12 11 10 9 8
RESERVED[12:5]
7 6 5 4 3 2 1 0
RESERVED[4:0] RST_WDW SWCLK_EXT USE
  • USE [read/write]: Windowing Selection
    Write 1 to use and 0 to bypass
  • SWCLK_EXT [read/write]: Use Windowing Swithing clock
    Write 1 to use windowing switching clock and 0
    to use internal generated clock
  • RST_WDW [read/write]: Reset Windowing module
    This effectivelly applies the windowing delay to
    its internal register.
    write 1: reset the module
    write 0: no effect
  • RESERVED [read-only]: Reserved
    Ignore on write, read as 0's
  • DLY [read/write]: Windowing Delay
    Write the number of ADC clock cycles to delay the
    start of windowing
